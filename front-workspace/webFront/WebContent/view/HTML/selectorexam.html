css 기본 선택자

선택자란?
특정한 HTML태그를 선택할떄 상요한느 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

모든 선택자

HTML 문사 안의 모든 태그를 선택할 때 전체 선택자를 사용할 수 잇다.

태그 선택자

HTML 문사 내에 같은 태그를 모두 선택할 때 사요안다.

-p {color : red;} 와 같은 형태로 사용 된다.

아이디 선택자

HTML 문서 내에 해당하는 아이디 속성의 값을 가진 태그만 선택 할때 사용한다.

- #아이디명 {color : blue;} 와 같은 형태로 사용 된다.

클래스 선택자

HTML 문서 내에 여러개의 태그를 동일한 클래스 명으로 지정하여 선택할 때 사용 한다.

- .클래스명 {color : purple;} 와 같은 형태로 사용 된다.
